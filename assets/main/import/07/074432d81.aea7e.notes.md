# 074432d81.aea7e.json UI Hide Notes

This file documents manual UI hides made in `074432d81.aea7e.json`.

Do not insert new fields into the compressed Cocos JSON arrays. For example, do not add `_active: false` or extra booleans into node arrays. That changes the serialized schema and can trigger runtime errors such as `m[n[0]] is not a function`.

The current hide method is:

- Set the node content size from `[5,width,height]` to `[5,0,0]`.
- Set the node transform scale values near the end of `_trs` to `0,0,0`.
- For text-only labels, clear the label string when needed.
